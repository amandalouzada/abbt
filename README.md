# Electron+Java Demo

Aplicação Java Desktop

## Bibliotecas e Frameworks

1. Node JS
2. Electron
3. Gradle
4. JDK 8
5. Jetty HTTP Server
6. jdbfReader

## Run aplicação


1. Download e instalação do `npm` from https://nodejs.org/en/download/

        > npm install

3. Build java:

        > ./gradlew build
        
4. Run java:

        > ./gradlew bootRun
        
5. Run electron:

        > npm start
